How It Works

Get Your API Key

Install the Extension

Download from GitHub Releases, run one command. Log out, log back in. Enable it. That's the whole installation.

Configure Your Preferences

Right-click the panel icon, paste your API key, pick your main language. Done. Extension is ready.

Quick Install
